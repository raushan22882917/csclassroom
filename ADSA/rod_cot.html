Dynamic Programming for Rod Cutting: Memoization and Bottom-Up Approaches

Introduction:

The rod cutting problem is a classic optimization challenge in computer science and dynamic programming. Given a rod of certain length and a price array corresponding to different lengths, the goal is to determine the maximum revenue that can be obtained by cutting and selling the rod. In this article, we explore two dynamic programming approaches to solve this problem: Memoization and Bottom-Up.

1. Memoized-Cut-Rod:

Memoization involves storing solutions to subproblems and reusing them to avoid redundant computations. The memoized_cut_rod function initializes an array r to store already computed solutions and then calls the helper function memoized_cut_rod_aux. This auxiliary function recursively calculates and memoizes the maximum revenue for different rod lengths. If the solution for a particular length is already stored in the r array, it is directly returned. Otherwise, the function calculates the maximum revenue by considering all possible cuts.


def memoized_cut_rod(p, n):
    r = [0] * (n + 1)
    return memoized_cut_rod_aux(p, n, r)

def memoized_cut_rod_aux(p, n, r):
    if r[n] > 0:
        return r[n]
    
    if n == 0:
        q = 0
    else:
        q = float('-inf')
        for i in range(1, n + 1):
            q = max(q, p[i] + memoized_cut_rod_aux(p, n - i, r))
    
    r[n] = q
    return q

2. Bottom-Up-Cut-Rod:

The Bottom-Up approach is an iterative technique that starts from smaller subproblems and builds up to the desired solution. The bottom_up_cut_rod function initializes an array r to store solutions for subproblems and iterates over rod lengths from 1 to n. For each length, it calculates the maximum revenue by considering all possible cuts and stores the result in the r array.


def bottom_up_cut_rod(p, n):
    r = [0] * (n + 1)

    for j in range(1, n + 1):
        q = float('-inf')
        for i in range(1, j + 1):
            q = max(q, p[i] + r[j - i])
        r[j] = q

    return r[n]

To demonstrate the usage of these algorithms, consider a price array prices = [0, 1, 5, 8, 9, 10, 17, 17, 20, 24, 30] and a rod length of 8. The memoized_cut_rod and bottom_up_cut_rod functions can be applied to find the maximum revenue achievable.


    prices = [0, 1, 5, 8, 9, 10, 17, 17, 20, 24, 30]
    rod_length = 8
    
    print("Memoized Cut Rod:", memoized_cut_rod(prices, rod_length))

Conclusion:

Dynamic programming provides efficient solutions to optimization problems like rod cutting. The Memoization and Bottom-Up approaches offer different perspectives on solving the same problem. While Memoization emphasizes recursive exploration with memorization of solutions, Bottom-Up builds solutions iteratively from smaller subproblems. Both techniques showcase the power and versatility of dynamic programming in tackling real-world challenges.
