Understanding Insertion Sort: A Simple Sorting Algorithm

Sorting algorithms are fundamental in computer science, used to arrange elements in a specific order. Insertion Sort is one such algorithm known for its simplicity and effectiveness, especially for small datasets. In this article, we'll explore how Insertion Sort works, discuss its implementation, and provide a clear example for better understanding.

Understanding Insertion Sort

Insertion Sort builds the final sorted array one element at a time, by repeatedly taking the next element and inserting it into its correct position in the already sorted part of the array. The main steps of the Insertion Sort algorithm can be summarized as follows:

  1. Iterate through the array, starting from the second element.
  2. Compare the current element with the elements to its left in the sorted part of the array.
  3. Shift elements to the right until finding the correct position for the current element.
  4. Insert the current element into its correct position.
  5. Repeat steps 2-4 until the entire array is sorted.

Example

Let's consider an array [5, 2, 4, 6, 1, 3] and walk through the Insertion Sort process:

  1. Initial array: [5, 2, 4, 6, 1, 3].
  2. After the first iteration, the array becomes [2, 5, 4, 6, 1, 3].
  3. After the second iteration, the array becomes [2, 4, 5, 6, 1, 3].
  4. After the third iteration, the array becomes [2, 4, 5, 6, 1, 3].
  5. After the fourth iteration, the array becomes [1, 2, 4, 5, 6, 3].
  6. After the fifth iteration, the array becomes [1, 2, 3, 4, 5, 6].

Explanation

Insertion Sort's simplicity lies in its approach of building the sorted array gradually. By iterating through the unsorted part of the array and inserting each element into its correct position in the sorted part, Insertion Sort achieves a time complexity of O(n^2) in the worst-case scenario. Although not as efficient as algorithms like Quick Sort or Merge Sort for large datasets, Insertion Sort performs well on small datasets and nearly sorted arrays.

Implementation

Here's a simple implementation of Insertion Sort in Python: 


def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

arr = [5, 2, 4, 6, 1, 3]
print("Original array:", arr)
insertion_sort(arr)
print("Sorted array:", arr)
